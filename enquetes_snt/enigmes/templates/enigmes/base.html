À propos

Cette plateforme est une application développée en Python avec le framework Django. Elle permet aux enseignants de SNT de créer, mutualiser et partager des énigmes à destination de leurs élèves. Le code source est disponible sur GitHub, où d'autres informations sur le projet sont disponibles. Une cagnotte a été ouverte afin de participer aux frais de fonctionnement de l'application.
