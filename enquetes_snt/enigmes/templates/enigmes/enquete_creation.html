Création d'une enquête

Un enquête consiste à sélectionner une ou plusieurs énigmes qui formeront l'enquête.

Vous avez deux possibilités pour créer une enquête :

  • Par sélection manuelle : vous parcourez les énigmes de la base et cochez celles vous intéressent
  • À partir d'une liste : vous renseignez les numéros des énigmes qui vous intéressent, ce qui constituera la clé de l'enquête

Pour chaque enquête, vous devrez :

  • renseigner une description de l'enquête (à destination uniquement de l'enseignant). Cette description peut contenir le nom de la classe ou du groupe concerné, le(s) thème(s) de l'enquête, etc. Cette description permettra de mieux vous y retrouver.
  • choisir si une correction est proposée aux élèves à la fin de leur enquête
  • choisir si les énigmes de l'enquête sont proposées dans un ordre aléatoire aux élèves

Attention, ces choix ne pourront pas être modifiés après la création de l'enquête. Dans ce cas, il faudra créer une nouvelle enquête, ce qui peut néanmoins se faire très rapidement avec la liste des numéros des énigmes.

Les enquêtes ne sont pas partagées avec les autres utilisateurs.

Pour chaque enquête, un code unique à 8 caractères sera généré. C'est ce code qu'il faut communiquer à vos élèves et qu'ils doivent saisir sur la page d'accueil de l'application pour démarrer l'enquête. Vous pourrez aussi leur communiquer une URL directe vers l'enquête.

Pour différencier vos différents élèves, un identifiant leur sera demandé. C'est à vous de définir les identifiants de vos élèves (un numéro par élève par exemple). Ces identifiants seront stockés dans la base de donnnées.

Vous retrouverez chaque enquête proposée ainsi que dans votre tableau de bord sur la page Espace personnel.

Par défaut, une enquête créée est active, c'est-à-dire qu'un élève muni du code de l'enquête peut lancer l'enquête pour la résoudre. Il est possible de la désactiver (et réactiver si besoin) via le tableau de bord, ce qui rendra son lancement impossible évidemment.

Vous pouvez également supprimer une enquête via le tableau de bord.
