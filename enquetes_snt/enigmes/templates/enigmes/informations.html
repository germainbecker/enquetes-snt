Création d'une enquête

Il y a deux façons de créer une enquête

  • par sélection manuelle : vous cochez les énigmes qui vous intéressent dans la liste des énigmes
  • par numéros d'énigmes : vous listez directement les numéros des énigmes qui vous intéressent

Au moment de la création d'une enquête, vous devrez également :

  • renseigner une description de l'enquête (à destination uniquement de l'enseignant). Cette description peut contenir le nom de la classe ou du groupe concerné, le(s) thème(s) de l'enquête, etc. Elle permettra de mieux vous y retrouver.
  • choisir si les élèves auront accès aux indications (si celles-ci existent pour les énigmes choisies)
  • choisir si une correction est proposée aux élèves à la fin de leur enquête
  • choisir si le score de l'élève lui est communiqué à la fin de son enquête (en cas de correction activée, le score est automatiquement affiché)
  • choisir si les énigmes de l'enquête sont proposées dans un ordre aléatoire aux élèves

Une enquête créée est par défaut active, c'est-à-dire qu'elle est disponible pour les élèves. Vous pouvez désactiver une enquête, elle ne sera alors plus accessible pour les élèves.

Diffusion aux élèves

Pour chaque enquête, un code unique à 8 caractères sera généré. C'est ce code qu'il faut communiquer à vos élèves et qu'ils doivent saisir sur la page d'accueil de l'application pour démarrer l'enquête. Vous pourrez aussi leur communiquer une URL directe vers l'enquête.

Le code de l'enquête est accessible dans le tableau de bord ou dans la page de détail de l'enquête. L'URL directe s'obtient en cliquant sur l'icône dans le tableau de bord ou dans la page de détail de l'enquête.

⚠️ IMPORTANT : les identifiants des élèves

Pour différencier vos différents élèves, un identifiant leur sera demandé.

C'est à vous de définir les identifiants de vos élèves. Un identifiant ne peut pas être vide et doit nécessairement être de la forme suivante :

  • une suite de caractères parmi les chiffres de 0 à 9 ainsi que les caractères "-" (tiret) et "_" (underscore).
  • éventuellement précédée d'une seule et unique lettre (optionnel, mais permet éventuellement de différencier les classes)

Par exemple, les identifiants 2569845, B25 et E-12_01 sont valides, mais 12/30 (caractère "/" non valide), 17B (lettre pas en première position) et A_25J (une deuxième lettre interdite) ne le sont pas.

Ces identifiants sont ainsi vierges de données personnelles sur les élèves. Ils seront stockés dans la base de données et permettront d'associer un résultat à chaque élève pour chaque enquête.

Résultats d'une enquête

Vous pouvez accéder aux résultats et réponses de vos élèves en cliquant sur le bouton dans le tableau de bord ou dans la page de détails de l'enquête.

La page de résultats d'une enquête permet de masquer les identifiants des élèves mais également leurs réponses et la correction. Elle contient également un récapitulatif des énigmes de l'enquête pour procéder à la correction devant les élèves. Le bouton "Actualiser" de cette page de résultats permet de mettre à jour en temps réel les réponses des élèves une fois qu'ils ont terminé l'enquête.

Il est aussi possible d'exporter les résultats d'une enquête dans un fichier CSV en cliqant sur le bouton dans le tableau de bord ou dans la page de détails de l'enquête ou dans la page de résultats de l'enquête. Ce fichier n'est généré que si au moins un élève a terminé l'enquête.

Modification d'une enquête

Vous pouvez modifier les énigmes choisies et les paramètres d'une enquête à tout moment, à condition de la désactiver au préalable. Il faudra alors la réactiver après les modifications pour que les élèves puissent y accéder.

Pour cela, il suffit de cliquer sur l'icône dans le tableau de bord ou directement dans la page de détail de l'enquête.

Duplication d'une enquête

Vous pouvez dupliquer une enquête, afin d'en créer une copie. La nouvelle enquête générée contient les mêmes énigmes et les mêmes paramètres que l'enquête de départ, et le mot "copie" s'ajoute automatiquement dans sa description. Cela peut être intéressant pour diffuser la même enquête (avec les mêmes énigmes mais avec un numéro d'enquête différent) à un autre groupe.

Pour cela, il suffit de cliquer sur l'icône dans le tableau de bord ou directement dans la page de détail de l'enquête.

Partage d'enquêtes entre collègues

Vous pouvez partager une enquête à d'autres collègues, simplement en leur communiquant l'URL de partage.

Pour obtenir cette URL, il suffit de cliquer sur l'icône dans le tableau de bord ou directement dans la page de détail de l'enquête.

Pour copier une enquête partagée sur son compte, il suffit d'être connecté à son compte et naviguer vers l'URL de partage. Avant de réellement copier l'enquête, il sera proposé un aperçu de celle-ci et la possibilité de modifier les paramètres de l'enquête, notamment sa description. Un nouveau code unique sera généré pour l'enquête copiée qui apparaîtra dans le tableau de bord.

Suppression d'une enquête

Vous pouvez également supprimer une enquête via le tableau de bord ou la page de détails de l'enquête, en cliquant sur le bouton . Cela supprimera l'enquête de la base de données ainsi que tous les résultats de celle-ci. Assurez-vous donc d'avoir récupéré les résultats avant la suppression.
